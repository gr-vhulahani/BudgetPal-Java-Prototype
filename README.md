# BudgetPal (Java Prototype)

## Overview

BudgetPal is a subscription-based personal finance coaching application designed to help people of all ages manage their money more effectively. Users manually record their income and expenses, while an AI-powered analysis engine reviews their spending habits each week and generates a personalised **Money Health Report** with three practical recommendations for improving their finances.

The application does not require bank account integration, making it accessible in any country and compatible with any currency.

## Features

* Manual income and expense tracking
* Weekly AI-generated Money Health Report
* Three personalised financial action tips every week
* Savings goal tracker
* Debt payoff planner
* Daily financial literacy lessons tailored to different age groups
* Multi-currency support
* No bank account linking required

## Technologies

* Java
* Object-Oriented Programming (OOP)
* Console-based prototype (Academic Project)

## Project Goal

This prototype demonstrates how Java can be used to build a personal finance coaching system that combines budgeting tools with AI-driven insights to encourage better financial habits. The focus is on accessibility, privacy, and financial education.
